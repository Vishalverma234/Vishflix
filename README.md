# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/Vishalverma234/Netflix-Clone.git
```

## Open Project

Simply open

```
index.html
```

or use **Live Server** in VS Code.

---

# 🔑 TMDB API Setup

1. Create an account on https://www.themoviedb.org/
2. Generate your API Key.
3. Open

```
js/home.js
```

Replace

```javascript
const API_KEY = "YOUR_API_KEY";
```

with your own TMDB API Key.
